## 🎨 Анимационные переходы

### Загрузка страницы
1. **Фоновая анимация** → появление звездного неба
2. **Пузырьки появляются** → scale(0) → scale(1) с задержкой
3. **Текст** → fade in снизу вверх
4. **Timeline** → slide in слева

### Взаимодействие с пузырьками
1. **Hover** → scale(1.1) + glow effect
2. **Click** → pulse animation + modal fade in
3. **XP gain** → floating +XP text с ease-out

### Смена уровней
1. **Level up** → screen flash + confetti
2. **Content unlock** → slide down with bounce
3. **Progress bar** → animated fill с sound effect

```
  ↓

  ↓

  ↓
50 XP → Level 3 Unlock (Contacts)

```

## 🎯 Key UX Принципы

1. **Progressive Disclosure** → информация открывается постепенно
2. **Gamification** → мотивация через XP и уровни
3. **Microinteractions** → каждое действие имеет обратную связь
4. **Mobile First** → адаптивный дизайн для всех экранов
5. **Accessibility** → keyboard navigation, screen readers
6. **Performance** → lazy loading, оптимизированные анимации

Этот UX flow превращает просмотр резюме в увлекательную игру! 🎮
